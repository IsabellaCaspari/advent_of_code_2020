# advent_of_code_2020

This is my code in kotlin for [https://adventofcode.com/](https://adventofcode.com/)


## installation
To run this code 

### Install Kotlin
```javascript
$ brew update
$ brew install kotlin
```

### Compile Application
```javascript
$ kotlinc 01_puzzel.kt -include-runtime -d puzzel1.jar
```

### Run the application
```javascript
$ java -jar puzzel1.jar
```
